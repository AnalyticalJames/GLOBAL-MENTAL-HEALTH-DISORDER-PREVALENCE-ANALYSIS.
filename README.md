Global Mental Health Disorder Analysis

This project explores global trends in mental health disorders using prevalence data across multiple countries and years. The dashboard highlights and shows how different mental health conditions vary in prevalence over time and across regions, supporting data driven discussions especially around public health awareness and policy planning.
This analysis focuses on clarity, simplicity, and responsible interpretation of sensitive health data.

Dataset
Source: Global Mental Health Disorder Dataset
Coverage: Multiple countries worldwide
Time Period: Up to 2019

Mental Health Disorders Included:

Depression
Anxiety disorders
Schizophrenia
Bipolar disorder
Eating disorders
Drug use disorders
Alcohol use disorders

Metric: Prevalence rate (proportion of population)

Data Preparation
I removed incomplete and error-prone records
Also i cleaned year and prevalence fields
Reshaped data from wide to long format using Unpivot
Ensured correct data types for numerical analysis too
Then finallly, I validated dataset integrity before visualization

Dashboard Features
Average prevalence comparison by mental health disorder
Trend analysis of mental health prevalence over tim
Country-level comparison for top affected regions
KPI cards summarizing key statistics
Interactive slicers for country, disorder, and year

Key Insights
Depression and anxiety disorders dominate global prevalence
Mental health prevalence shows a gradual increase over time
Certain disorders remain consistently low in prevalence
Mental health burden varies significantly by country
Small percentage differences represent large population impacts
